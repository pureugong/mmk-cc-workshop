# /summarize-video — YouTube 영상 자막 추출 및 요약

YouTube 영상의 자막을 추출하고 한국어로 요약합니다.

## 인자

$ARGUMENTS — YouTube 영상 URL (예: https://www.youtube.com/watch?v=VIDEO_ID)

## 실행 절차

1. `mmk youtube metadata $ARGUMENTS` 를 실행하여 영상 메타데이터(제목, 채널, 길이 등)를 확인합니다.

2. `mmk youtube transcript $ARGUMENTS` 를 실행하여 자막 텍스트를 가져옵니다.
   - 자막이 없으면 사용자에게 알리고 종료합니다.

3. 가져온 자막을 분석하여 **한국어로** 다음 형식으로 요약합니다:

### 요약 형식

```
## 📝 영상 요약

**제목**: {영상 제목}
**채널**: {채널명}
**URL**: {영상 URL}

### 한줄 요약
{영상의 핵심 내용을 한 문장으로}

### 주요 내용
- {핵심 포인트 1}
- {핵심 포인트 2}
- {핵심 포인트 3}
- {핵심 포인트 4}
- {핵심 포인트 5}

### 키워드
{관련 키워드 3-5개, 쉼표 구분}
```

## 주의사항
- 자막이 영어나 다른 언어여도 반드시 **한국어**로 요약합니다.
- 요약은 3-5개 bullet point로 핵심만 간결하게 작성합니다.
- 기술적 용어는 원어를 병기합니다 (예: 프롬프트 엔지니어링(Prompt Engineering)).
