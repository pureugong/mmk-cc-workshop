# /fetch-videos — YouTube 채널 새 영상 감지

등록된 YouTube 채널들의 최근 영상을 확인하고 아직 처리하지 않은 새 영상 목록을 보여줍니다.

## 실행 절차

1. `python3 src/fetch_videos.py` 를 실행하여 모든 채널의 최근 영상 목록을 가져옵니다.
2. 가져온 JSON 결과를 `python3 src/state.py filter` 에 파이프하여 이미 처리한 영상을 제외합니다.
   ```bash
   python3 src/fetch_videos.py | python3 src/state.py filter
   ```
3. 결과를 사용자에게 보기 좋게 정리하여 보여줍니다:
   - 새 영상이 있으면: 각 영상의 제목, 채널명, 게시일, URL을 테이블 형태로 표시
   - 새 영상이 없으면: "새 영상이 없습니다" 메시지 출력

## 출력 형식 예시

```
📺 새 영상 2개 발견

| # | 채널 | 제목 | 게시일 | URL |
|---|------|------|--------|-----|
| 1 | 채널1 | 영상 제목... | 2025-01-15 | https://... |
| 2 | 채널2 | 영상 제목... | 2025-01-14 | https://... |
```
