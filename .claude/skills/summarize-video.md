# /summarize-video - YouTube 영상 자막 추출 및 요약

YouTube 영상의 자막을 추출하고 핵심 내용을 요약합니다.

## 인자

- `<youtube-url>`: 요약할 YouTube 영상 URL (필수)

예시: `/summarize-video https://www.youtube.com/watch?v=xxxxx`

## 실행 절차

### 1. 영상 메타데이터 확인

```bash
mmk youtube metadata <youtube-url>
```

영상 제목, 채널명, 업로드일 등 메타데이터를 확인합니다.

### 2. 자막 추출

```bash
mmk youtube transcript <youtube-url>
```

자막을 텍스트로 추출합니다. 자막이 없는 경우 사용자에게 알립니다.

### 3. 핵심 요약 생성

추출된 자막을 분석하여 다음 구조로 요약합니다:

```markdown
## 📊 [영상 제목]
**채널**: [채널명] | **날짜**: [업로드일]

### 핵심 요약
- (3~5줄 핵심 내용 요약)

### 주요 포인트
1. (구체적인 시장 분석 / 경제 이슈 포인트)
2. ...
3. ...

### 언급된 종목/지표
- (영상에서 언급된 주요 종목, 지수, 경제 지표 리스트)

### 투자 시사점
- (투자자 관점에서의 시사점 1~2줄)
```

**요약 원칙:**
- 한국어로 요약
- 수치/데이터는 최대한 구체적으로 포함
- 전문 용어는 그대로 유지
- 개인적 의견이 아닌 영상 내용 기반으로만 요약
- 불필요한 인트로/아웃트로 내용은 제외

### 4. 결과 출력

요약 결과를 사용자에게 보여줍니다.
