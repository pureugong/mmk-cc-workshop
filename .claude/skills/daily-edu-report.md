# daily-edu-report

매일 1회 실행되는 통합 교육 인사이트 리포트 스킬입니다.
3가지 단계를 순서대로 실행합니다.

## 실행 흐름

### 1단계: 인사이트 생성

WebSearch와 mmk CLI를 활용하여 최신 한국 교육 인사이트를 생성합니다.

1. **WebSearch**로 아래 키워드들을 검색합니다 (최소 3개):
   - "한국 교육 정책 {현재연도}"
   - "교육과정 변화 트렌드"
   - "학부모 교육 고민 {현재연도}"
   - "교사 업무 환경 변화"
   - "입시 제도 변화"
   - "AI 교육 활용"

2. 검색 결과에서 교육 관련 **YouTube 영상 URL**이 있으면 `mmk youtube transcript <url>` 로 자막을 추출합니다. 없으면 웹 검색 결과만으로 진행합니다.

3. 수집한 정보를 바탕으로 3가지 관점의 인사이트를 작성합니다:
   - **교육의 방향**: 한국 교육의 큰 흐름, 정책 변화, 미래 방향 (3~5문장)
   - **학부모/학생 관점**: 학부모가 알아야 할 변화, 학생에게 미치는 영향 (3~5문장)
   - **선생님 관점**: 교사 역할 변화, 수업/평가 방식 변화, 필요 역량 (3~5문장)
   - 적절한 **카테고리** 선정: 정책/트렌드/입시/교육과정/교육환경
   - **출처 URL** 1개 이상 확보
   - 간결한 **제목** 작성

### 2단계: Notion 저장

`notion-create-pages` MCP 도구로 Notion 데이터베이스에 저장합니다.

- **Parent**: `{"type": "data_source_id", "data_source_id": "d63781b2-859b-4f27-b213-9d48bc1e5c46"}`
- **Properties**:
  - `"제목"`: 인사이트 제목
  - `"date:날짜:start"`: 오늘 날짜 (ISO-8601, 예: "2026-04-09")
  - `"교육방향"`: 교육 방향 인사이트 텍스트
  - `"학부모/학생 관점"`: 학부모/학생 관점 텍스트
  - `"선생님 관점"`: 선생님 관점 텍스트
  - `"출처"`: 대표 출처 URL
  - `"카테고리"`: 정책/트렌드/입시/교육과정/교육환경 중 하나
  - `"상태"`: "완료"
- **Content**: 3가지 관점을 마크다운으로 정리한 본문

### 3단계: Slack 전송

`slack_send_message` MCP 도구로 Slack에 알림을 보냅니다.

- **channel_id**: `U05BCDYR4KC` (DM, 채널 생성 시 해당 채널 ID로 변경)
- **메시지 포맷**:
```
*일일 교육 인사이트* — {YYYY-MM-DD}

*교육의 방향*
{교육 방향 인사이트}

*학부모/학생 관점*
{학부모/학생 관점 인사이트}

*선생님 관점*
{선생님 관점 인사이트}

카테고리: {카테고리}
출처: {URL}
```

## 스케줄링

이 스킬을 매일 자동 실행하려면:
```
/loop 24h /daily-edu-report
```
