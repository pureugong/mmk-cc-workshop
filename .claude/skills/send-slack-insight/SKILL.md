# send-slack-insight

생성된 교육 인사이트를 Slack으로 전송하는 스킬입니다.

## 설정

- **채널 ID**: `U05BCDYR4KC` (현재 DM으로 설정됨)
- 채널을 새로 만든 경우 해당 채널 ID로 변경하세요

## 실행 방법

1. `slack_send_message` MCP 도구를 사용합니다.
2. `channel_id`에 위 채널 ID를 지정합니다.
3. 아래 메시지 포맷에 맞춰 전송합니다.

## 메시지 포맷

```
*일일 교육 인사이트* — {오늘 날짜 YYYY-MM-DD}

*교육의 방향*
{교육 방향 인사이트 내용}

*학부모/학생 관점*
{학부모와 학생 관점 인사이트}

*선생님 관점*
{교사 관점 인사이트}

카테고리: {정책/트렌드/입시/교육과정/교육환경}
출처: {기사/영상 링크}
```

## 주의사항

- Slack 메시지는 최대 5000자 제한이 있으므로 간결하게 작성
- 마크다운은 Slack 스타일 사용: `*bold*`, `_italic_`, `` `code` ``
- 링크는 일반 URL로 포함
